# wechat-decrypt - 微信数据解密工具 - 项目规划

## 项目概述
Python 3 + MCP (Model Context Protocol) + SQLite 解密工具，用于解密和分析微信数据库，支持图片解码、数据库解密、密钥查找，包含 MCP 服务器和监控 Web 界面。

---

## 2026-03-03 Claude 建议

### 现状评估
工具功能完整（解密→分析→MCP集成），MCP 服务器集成是创新点，可以让 AI 助手直接访问微信数据。与 `20260301-WechatBackup` 等项目共同构成微信数据处理生态。

### 功能建议

#### 短期（工具完善）
1. **解密可靠性**
   - 支持微信最新版本（密钥位置可能变化）
   - Windows 11 + 微信3.x/4.x 兼容性测试
   - macOS 支持（LLDB 解密方案）
   - 批量解密（多个数据库文件）

2. **数据解析**
   - 更多消息类型（视频号/引用消息/小程序）
   - 时间戳转换（微信特有时间格式）
   - 多媒体文件关联（消息与图片/视频文件对应）
   - 联系人信息整合（人名映射）

3. **MCP 服务器增强**
   - 提供更丰富的 MCP 工具集：
     - `search_messages`: 搜索消息
     - `get_contact_messages`: 获取某联系人的所有消息
     - `get_media`: 获取媒体文件
     - `export_chat`: 导出聊天记录
   - MCP 服务器文档（供 Claude/Cursor 等 AI 工具配置）

4. **监控界面**
   - Web 界面优化（更好的数据展示）
   - 实时监控（新消息提醒）
   - 聊天记录可视化（时间线布局）

#### 中期
1. **与 `20260301-WechatBackup` 整合**
   - 解密层：此项目（wechat-decrypt）
   - 查看层：WechatBackup（更完善的查看界面）
   - 两个项目各司其职，通过 API 连接

2. **数据分析**
   - 聊天频率分析
   - 词频统计
   - 关系图谱（和谁交流最多）

3. **隐私保护**
   - 本地处理保证（明确声明不上传任何数据）
   - 处理完成后清理临时文件
   - 敏感信息脱敏导出选项

#### 长期
1. **微信数据生态整合**
   - 整合仓库中所有微信相关项目
   - 统一的"微信数据管理套件"
   - 统一安装脚本

2. **AI 分析**
   - 通过 MCP 让 AI 直接分析聊天记录
   - 生成聊天摘要/年度报告
   - 识别重要信息（待办/承诺/约定）

### 技术建议
- MCP SDK 版本保持最新（Claude Desktop 定期更新）
- 添加 MCP 配置示例（`mcp_config.json` 格式）
- 注意合法性：仅解密/分析自己账号的数据
- 添加使用声明（免责条款）
